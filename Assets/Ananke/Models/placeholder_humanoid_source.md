# Placeholder humanoid source

This repository includes `placeholder_humanoid.fbx`, a deliberately simple ASCII FBX placeholder mesh.

- License: CC0 / public domain dedication by the repository authors.
- Purpose: provide a zero-dependency stand-in humanoid silhouette for the demo scene until a final rigged mesh is chosen.
- Structure: three stacked boxes approximating legs, torso, and head so Unity importers can substitute a real humanoid later without changing the scene layout.
